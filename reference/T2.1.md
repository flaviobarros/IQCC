# Hotelling T² Statistic for Phase I

Calculate the Hotelling T² statistic for Phase I using estimated mean
vector and covariance matrix. For individual observations (\\n = 1\\):
\\T^2 = x_i' S^{-1} x_i\\. For subgroups (\\n \> 1\\): \\T^2_i = n
(\bar{x}\_i - \bar{\bar{x}})' S^{-1} (\bar{x}\_i - \bar{\bar{x}})\\.

## Usage

``` r
T2.1(estat, m, n)
```

## Arguments

- estat:

  A list returned by
  [`stats`](https://flaviobarros.github.io/IQCC/reference/stats.md) with
  three components: `[[1]]` grand mean vector, `[[2]]` pooled covariance
  matrix, `[[3]]` matrix of subgroup means (\\m \times p\\).

- m:

  Number of subgroups (Phase I sample size).

- n:

  Subgroup size. Use \\n = 1\\ for individual observations, \\n \> 1\\
  for subgroups.

## Value

A numeric vector of length \\m\\ with the T² statistics.

## Details

Before using this function it is necessary to execute
[`stats`](https://flaviobarros.github.io/IQCC/reference/stats.md) (that
calculates the auxiliary statistics involved in the T² formula) and the
function
[`data.1`](https://flaviobarros.github.io/IQCC/reference/data.1.md) (or
other way to supply the data).

## Phase convention

Phase I `---` uses parameters estimated from the same data being
plotted. The T² statistics are not independent.

## Errors

Stop if \\n \< 1\\.

## Decision rule

A Phase I point signals when T² exceeds the UCL from
[`cchart.T2.1`](https://flaviobarros.github.io/IQCC/reference/cchart.T2.1.md).

## References

Montgomery, D.C., (2009). "Introduction to Statistical Quality Control".
Chapter 11. Wiley.
