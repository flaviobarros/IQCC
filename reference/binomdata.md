# Binomial Process Data for p-Charts

25 subgroups with sample sizes (`ni`), number of nonconforming units
(`Di`), and proportion nonconforming (`pi`). This data set is used for
\\p\\-chart examples.

## Format

A data frame with 25 observations on the following 4 variables.

- i:

  Index.

- ni:

  The sample Size.

- Di:

  Number of non-conforming units per sample.

- pi:

  Proportion of non-conforming units per sample.

## Source

Montgomery, D. C. (2009). *Introduction to Statistical Quality Control*,
6th ed. Wiley.

## Examples

``` r

data(binomdata)
```
