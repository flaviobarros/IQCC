Builds statistical control charts with exact limits for univariate and
multivariate cases.

## Installation

You can install from CRAN:

``` r
install.packages('IQCC', dependencies = TRUE)
```

or from github:

``` r
devtools::install_github('flaviobarros/IQCC')
```
